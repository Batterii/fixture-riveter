# fixture-riveter

fixture-riveter is a fixtures replacement library based on thoughtbot's excellent ruby library factory_botopen in new window, with some ideas adapted from factory-girlopen in new window, another javascript library inspired by factory_bot. It's written in Typescript, is fully async, and implements nearly all of factory_bot's features.

It was built for a knex/objection.js codebase and thus the integration with that ORM is the best. It has support for other ORMs through the use of adapters, but some of the features don't work as well or at all with them.

# Contributing
